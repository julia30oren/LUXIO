## Navigation
1. FrontEnd **Luxio-app**
2. BackEnd **server**
3. Dev **dev-apps**

## Installation
- Open **Luxio-app** folder
    - run *npm i*
- Open **server** folder
    - run *npm i*
- Open **dev-apps** folder
    - run *npm i*
    - run *npm start*
