- BackEnd **server**
- Dev **dev-apps**

## Installation
1. Open **Luxio-app** folder
    - run *npm i*
2. Open **server** folder
    - run *npm i*
3. Open **dev-apps** folder
    - run *npm i*
    - run *npm start*
